# 缺陷报告：游客首页磁贴跳转微信 / 外部登录（离开应用）

| 字段 | 值 |
|---|---|
| **产品** | 通信工程师考试 |
| **包名** | `com.csai.tongxin` **v1.0.1** |
| **严重程度** | 中（UX / 非预期外部跳转） |
| **平台** | HarmonyOS NEXT |

## 摘要
在**未登录首页**上，多个磁贴（如 **去做题**，以及此前的 **测试记录 / 错题本 / 试题收藏 / 电子资料**）会将用户带出应用，进入 **微信**（`com.tencent.wechat`）手机号登录 UI，而非考试应用内的登录浮层或明确的“需要登录”状态。

## 步骤
1. 冷启动应用；完成考试选择 + 隐私（若出现）；在**未登录**状态下到达首页。
2. 点击 **去做题**（历年真题卡片）— 或 **测试记录 / 错题本**（当这些目标可点时）。
3. 观察前台包名 / UI。

## 预期
- 应用内登录弹窗（其他入口曾出现希赛短信登录），**或**
- 磁贴禁用 + 提示“请先登录”，**或**
- 游客可访问的内容。

## 实际
- 进程离开 `com.csai.tongxin` FOREGROUND。
- 出现系统/微信 **“用手机号登录” / 同意并登录** UI。
- 用户须手动返回；易被感知为崩溃或劫持。

## 证据（实验室）
- 点击后 `aa dump`：FOREGROUND `com.tencent.wechat`
- 截图判定：在 `uitest uiInput click 300 1200`（去做题区域）上内容变化 + 包名切换

## 备注
- 与冷启动白屏缺陷不同。
- 可能是有意的 SSO — 若游客首页将“去做题”标为可用，仍属产品缺陷。
